# Markdown Editor

A modern, feature-rich Markdown editor with real-time preview functionality.

## Features

- **Live Preview**: See your Markdown render instantly as you type
- **Clean Modern UI**: Dark theme with smooth animations and gradient accents
- **Full Markdown Support**: 
  - Headings (H1-H6)
  - Lists (ordered and unordered)
  - Code blocks with syntax highlighting
  - Tables
  - Links and images
  - Blockquotes
  - Horizontal rules
- **Responsive Design**: Works beautifully on desktop and mobile devices
- **Text Statistics**: Real-time count of lines and words
- **Keyboard Shortcuts**: 
  - Ctrl/Cmd + A: Select all
  - Ctrl/Cmd + S: Save (placeholder functionality)
- **Quick Actions**:
  - Copy content to clipboard
  - Clear all content
- **Smooth Animations**: Professional hover and transition effects

## Installation

1. Clone the repository
2. Open `index.html` in your preferred web browser

## Usage

1. Type your Markdown content in the left editor pane
2. See the real-time preview in the right pane
3. Use the header buttons to copy or clear content
4. Monitor text statistics in the status bars at the bottom of each pane

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, gradients, and animations
- **JavaScript**: Core functionality
- **Marked.js**: Markdown parsing and rendering

## Browser Support

Works on all modern web browsers that support ES6+ features:

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
